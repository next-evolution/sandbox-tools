# 経済指標コード命名規則

`data/economic_indicator-all.csv` の `code` 列（`name` 列から人手で採番した英語コード）の命名パターンをまとめる。
新しい指標行を追加する際はこのルールに従う。

---

## 構成要素

`code` は基本的に次の順で連結する。

```
{指標本体}_{改定区分}_{期間比較}
```

例: `GDP_R_QOQ` = GDP（改定値）（前期比）

---

## 1. 指標本体の略称

一般的な経済指標の英語略称を使う。

| 日本語 | code |
|---|---|
| 国内総生産 | `GDP` |
| 消費者物価指数 | `CPI` |
| 卸売物価指数・生産者物価指数 | `PPI` |
| 購買担当者景気指数 | `PMI` |
| 鉱工業生産 | `IPI` |
| 失業率 | `UNEMP` |
| 小売売上高 | `RETAIL` |
| 貿易収支 | `TRADE` |
| 個人消費支出 | `PCE` |

## 2. 中央銀行・要人発言の略称

`name` に含まれる中央銀行名・総裁名は、金融業界で広く使われる機関の英語略称に変換する。**人名（パウエル、ラガルド、ベイリー等）は落とし、機関略称のみ採用する。**

| 国・地域 | 中央銀行 | code |
|---|---|---|
| 豪 | 豪準備銀行 | `RBA` |
| 伯 | ブラジル中央銀行 | `BCB` |
| 加 | カナダ銀行 | `BOC` |
| 瑞 | スイス国立銀行 | `SNB` |
| 欧 | 欧州中央銀行 | `ECB` |
| 英 | イングランド銀行 | `BOE` |
| 日 | 日本銀行 | `BOJ` |
| 米 | 連邦準備制度 | `FED` |
| 印 | インド準備銀行 | `RBI` |
| 墨 | メキシコ銀行 | `BANXICO` |
| 諾 | ノルゲスバンク | `NORGES` |
| 新 | ニュージーランド準備銀行 | `RBNZ` |
| 露 | ロシア中央銀行 | `CBR` |
| 瑞典 | リクスバンク | `RIKSBANK` |
| 土 | トルコ中央銀行 | `TCMB` |
| 南ア | 南アフリカ準備銀行 | `SARB` |

例: 「英中銀(BOE)総裁発言」→ `BOE_SPEECH`、「米連邦準備理事会(FRB)議長定例記者会見」→ `FED_PC`

**注意:** 総裁・議長個人の名前（ベイリー、パウエル、ラガルド等）は `name` に含めない。人事異動で総裁が交代しても指標の実体（総裁発言・記者会見）は変わらないため、機関名のみで表記する。

## 3. 改定区分・期間比較のサフィックス

| サフィックス | 意味 |
|---|---|
| `_MOM` | 前月比 |
| `_YOY` | 前年（同月）比 |
| `_QOQ` | 前期比 |
| `_ANN` | 年率換算 |
| `_Q` | 四半期 |
| `_R` | 改定値 |
| `_F` | 速報値 |
| `_FF` | 確定値 |
| `_CORE` | コア指数（除く食品・エネルギー等） |
| `_EX_AUTO` | 除く自動車 |
| `_SA` | 季節調整済 |
| `_NSA` | 季節調整前 |

複数該当する場合は上記の順序（指標本体 → 改定区分 → 期間比較）で連結する。
例: `GDP_R_QOQ`（GDP改定値・前期比）、`CPI_CORE_YOY`（CPIコア指数・前年同月比）

## 4. ニュース系コード（`NEWS_INFO` / `NEWS_SHOCK`）

定期発表される経済指標とは別に、不定期・速報で発生し相場に影響し得るニュースは `data/economic_indicator.csv` に `NEWS_INFO` / `NEWS_SHOCK` として登録する。どちらも `country_code` ごとに20行（`data/country.csv` の全国コード分）を用意する。

| code | importance | 用途 |
|---|---|---|
| `NEWS_INFO` | `X` | 相場に影響する不定期ニュース全般（為替介入、要人発言・解任懸念、選挙結果、関税報道、財政不安報道など） |
| `NEWS_SHOCK` | `Z` | 相場の急騰急落に多大な影響を及ぼす、世界的インパクトを持つ事象の**発生・開始**（戦争・軍事攻撃の開始など） |

### 使い分けの基準

- **「歴史的に開始された事象」= `NEWS_SHOCK`**：戦争・侵攻・軍事攻撃が発生した最初の報道。米国イラン攻撃、ロシアのウクライナ侵攻級。
- **「継続中の関連・追加報道」= `NEWS_INFO`**：`NEWS_SHOCK` に該当する事象が始まった後の、状況変化・延長・緩和などの継続報道。

`data/economic_indicator_data.csv` での実例:

| memo | 判定 | 理由 |
|---|---|---|
| ロシア-ウクライナ侵攻 | `NEWS_SHOCK` | 侵攻という事象の開始 |
| イラン攻撃 | `NEWS_SHOCK` | 攻撃という事象の開始 |
| ベネズエラ攻撃 | `NEWS_SHOCK` | 攻撃という事象の開始 |
| イラン攻撃5日間延長報道 | `NEWS_INFO` | イラン攻撃という事象の継続報道 |
| 米軍ホルムズ海峡逆封鎖示唆 | `NEWS_INFO` | 同上、状況変化の継続報道 |
| イラン外相ホルムズ海峡解放 | `NEWS_INFO` | 同上、事態緩和の継続報道 |

迷った場合は「これが世界初のニュース速報として流れた瞬間か」を基準に判定する。それ以降の関連報道は `NEWS_INFO` にとどめる。

| サイト | 用途 |
|---|---|

| [Investing.com 経済指標カレンダー](https://www.investing.com/economic-calendar/) | 日本語名と英語名が併記され対応が取りやすい |
| [Forex Factory Calendar](https://www.forexfactory.com/calendar) | MOM/YOY/QOQ 等の略記が業界標準に近い |
| 各国中央銀行公式サイト（BOE, ECB, Fed, BOJ 等） | 総裁名・機関略称の一次情報 |
